# Survey
# 📊 Enquête d'Opinion Dynamique

Un générateur d'enquêtes multi-étapes moderne, responsive et paramétrable.

## ✨ Fonctionnalités
- **Génération dynamique** : Modifiez le tableau `surveyConfig` pour changer les questions.
- **Multi-étapes** : Interface fluide une question par une.
- **Sauvegarde automatique** : Utilise le `localStorage` pour ne pas perdre la saisie.
- **Mode Sombre** : Détection automatique des préférences système.
- **Réception par Email** : Intégration directe via FormSubmit.
- **Responsive Design** : Utilisable sur mobile, tablette et desktop.

## 🚀 Installation
1. Clonez ce dépôt.
2. Remplacez `TON_EMAIL@MAIL.COM` dans `script.js` par votre adresse.
3. Ouvrez `index.html` ou hébergez-le sur GitHub Pages.
